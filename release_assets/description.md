# Google Play Store Listing

## App Name
Endless Runner

## Short Description (Max 80 chars)
Run, dodge, and survive in this high-speed infinite 3D runner!

## Long Description (Max 4000 chars)
**Endless Runner** is an adrenaline-pumping 3D arcade game where your only goal is to survive as long as possible!

**Features:**
*   **Infinite Gameplay**: Properties randomly generated world that never ends.
*   **High-Speed Action**: Test your reflexes as the speed increases.
*   **3D Graphics**: Immersive visual experience.
*   **Challenging Obstacles**: Dodge walls, barriers, and traps.

How far can you run before the inevitable crash? Challenge yourself and beat your high score in **Endless Runner**!

*   **App Icon**: 512 x 512 px (32-bit PNG)
*   **Feature Graphic**: 1024 x 500 px (JPG or 24-bit PNG, no alpha)
